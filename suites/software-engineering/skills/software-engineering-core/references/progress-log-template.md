# Progress Log Template

Use this when work spans multiple steps, changes direction, uncovers new evidence, or needs interruption-safe state.

```md
# Progress Log: <task name>

## Timeline

## Evidence Inspected

## Decisions Made

## Changes Made

## Verification Runs

## Intent Conformance

- Intended state
- Observed state
- Material deltas
- Verdict: `conforms`, `authorized deviation`, or `unresolved deviation`

## Proof Gaps

## Deviations

Record material departures when they occur, including corrected departures. Write `none observed` only after comparison.

## Next Action
```
